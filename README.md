# YUMMY_Online_DineIn_Reservation
The YUMMY online dine-in reservation system is a web application that allows customers to reserve a table at their favourite restaurant online. The system is built using HTML, CSS, JavaScript, PHP, and MySQL.


FEATURES 

ADMIN PANEL – YUMMY

•	Secure Login/Logout
•	Dashboard Page
•	Restaurant Registration
•	List all Restaurants and Dine in Details
•	View Dine in Details
•	Manage System Information
•	Manage Account Credentials


CLIENT-SIDE (RESTAURANT)

•	Secure Login/Logout
•	Dashboard Page
•	Add or edit banners and offers
•	Add or edit Food items
•	View booking details
•	Check-in conformation 
•	View payments 
•	Manage Account Credentials


CUSTOMER-SIDE

•	Secure Login and Registration
•	Explore Restaurants
•	Filter Food by selected/checked items 
•	List Food by Categories
•	List Food by Sub Categories
•	View Food and Restaurants
•	Booking Restaurants
•	Manage Booking
•	Confirmation of booking and paying using a Cart or other payment methods
•	View Booking Details


TECHNICAL USED

FRONT-END
•	HTML
•	CSS
•	JavaScript

BACK-END
•	PHP
•	MySQL

SOFTWARE USED
•	XAMPP
•	Visual Studio Code
•	Web Browser (Google Chrome)
